## Key Components

**Frontend:**
- React Hooks (useState, useRef, useCallback)
- Webcam component (react-webcam)
- File input handling
- Fetch API for backend calls
- Dynamic state management

**Backend:**
- FastAPI route handlers
- Multipart form data parsing
- Image I/O with PIL
- Face detection algorithms
- Base64 encoding

**AI Engine:**
- Face detection (Haar Cascade or simple positioning)
- Image resizing and alignment
- Alpha mask creation
- Image blending/compositing
